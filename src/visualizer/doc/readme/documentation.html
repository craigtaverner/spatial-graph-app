API Documentation and Developer's Guide

The Documentation Viewer is located in the directory doc/api. It can be used to browse and search both the API documentation and the Developer's Guide.

The introduction videos page lists tutorials and screencasts about working and developing applications with yFiles for HTML.

The Developer's Guide describes in detail the architecture and all main features of yFiles for HTML.

Getting Started Information

The Getting Started chapter of the Developer's Guide guides you through the first steps.

Source Code Demos and Tutorials

Extensive source code demo applications and source code tutorials are located in the demos directory of the yFiles for HTML package.

In addition, these demos are hosted at live.yworks.com and their source code is available at the yfiles-for-html-demos GitHub project. Note that you're not allowed to upload the yFiles for HTML redistributables to GitHub or any other file sharing site.

Demo Server

Due to the strict cross domain policies of some browsers, some demos do not work correctly when launched from the filesystem in these browsers. In particular, this applies to all demos that load GraphML files, for example as initial graph.

Therefore, we recommended to access the demos through a Web server. Install and start scripts for a local Node.js Express server are included. To use that server, run the following commands in a terminal: 

            > cd %YFILES_HTML_DISTRIBUTION_DIR%/demos/demo-server/
            > npm install
            > npm start
